# Imedpharm SIA Website Upgrade

A responsive static website for Imedpharm SIA, a pharmaceutical wholesale company.

## Structure

- `index.html` — main landing page (English)
- `lv.html` — Latvian version
- `ru.html` — Russian version
- `styles.css` — responsive styling
- `script.js` — mobile navigation behavior

## Usage

1. Open `index.html` in a browser.
2. Edit the content in `index.html` to match the final company copy.
3. Replace any placeholder images or assets as needed.

## Notes

- The site is designed as a lightweight, static company landing page.
- Add additional pages or translate content as needed for multi-language support.
